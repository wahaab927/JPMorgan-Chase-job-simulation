# JPMorgan-Chase-Analytics


This repository contains work completed as part of the JPMorgan Chase Virtual Experience 

## Task 1: Natural Gas Price Analysis
- Analyzed historical monthly natural gas prices
- Built a pricing model to estimate gas prices for any historical date
- Extrapolated prices one year into the future using trend and seasonality
- Implemented the solution in Python using regression and time-series concepts

## Tools & Skills
- Python
- Pandas, NumPy
- Time Series Analysis
- Regression Modeling
- Data Visualization
